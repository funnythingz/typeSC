This is Project Template for TypeScript & Sass/Compass.

## Getting Started

Install Grunt Plugin

```shell
% npm install
```

### Directory

#### Dev

```shell
src
├── html
│   └── index.html
├── scss
│   └── style.scss
└── ts
    └── oop.ts
```

#### build

```shell
build
├── app.js
├── app.min.js
├── index.html
└── style.css
```

### Command

#### compile build & open & watch

```shell
% grunt
```

#### build only

```shell
% grunt build
```
